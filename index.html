Czym jest frontend?

frontend/front end/ front-end -> all are correct

Frontend - to co się wyświetla w przeglądarce, youtube - aplikacja frontendowa, też się wyświetla w przeglądarce, spotify, VSC - edytor stworzony za pomocą technologii frontendowych ale nie przeglądarkowy.
HTML, CSS, JavaScript - technologie używane do budowania aplikacji, stron.

Frontend:

  • odpowiada za wygląd stron/aplikacji internetowych
  • odpowiada za interakcje z użytkownikiem
  • generowany przy użyciu 3 technologii (głównie) HTML, CSS, JavaScript (WebAssembly - nie jest jeszcze super popularne - pozwala na uruchamianie kodu nie napisanego w JS)
  • technologie frontendowe przejmują rynek tradycyjnych aplikacji (w wybranych obszarach)

Ścieżka rozwoju

Taka fajna mapka

Warstwy strony internetowej:

  • 0 treść, kontent, wszystko co jest nośnikiem informacji (tekst, link, lista, video, obrazki, formularze, definicje, nagłówki)
  • 1 Semantyka HTML, opakowuje treść w znaczniki
  • 2 Prezentacja CSS, nadaje wygląd elementom
  • 3 Interakcja: JavaScript

Opowieść o wężu, kebabie, wielbłądzie i Pascalu

Nazywanie rzeczy:

  • bez polskich znaków
  • bez spacji
  • angielskie słowa
  • jeden schemat konsekwentnie

Konwencje nazewnictwa:

  • camelCase - superSweetKiss (najczęściej)
  • kebab-case - super-sweet-kiss (najczęściej)
  • snake_case - super_sweet_kiss
  • PascalCase - SuperSweetKiss

Kanban

Planowanie pracy w celu śledzenia swoich postępów np. nauki.

Kanban - podział na 3 kolumny (lub więcej):

  • TODO = do zrobienie (dobrze opisać definition of done)
  • DOING = w trakcie (najlepiej jedno zadanie na raz)
  • DONE = zrobione

Korzystaj z metody Kanban na Trello.com

GitHub

Możliwość pokazywania kodu na zewnątrz, taka nasza programistyczna wizytówka, portfolio. GitHub GitHub.io

Tworzymy nowe repozytorium - zdalne, dodajemy tylko nazwę najlepiej taką samą jak katalog. Chcemy dodać nasze z dysku, wklejamy to co nam podpowiada GitHub do terminala, logujemy się.
Pliki wysłały się do zdalnego repozytorium na GitHubie.
